# Hardware

Complete hardware specifications for the homelab.

## Compute

### IBM Server

- **Processors:** 2x Intel Xeon E5-2650 (8 Core, 16 Threads each)
- **RAM:** 384GB DDR3 ECC 1600MHz
- **Storage:** 8x 2.5" Toshiba 3.2TB SAS SSD (25.6TB total)
- **Power Supply:** 2x 750W 80 Plus Platinum PSUs
- **Purpose:** Primary compute and storage server

### Ubuntu Server

- **Processor:** Intel Core i7-9700K
- **RAM:** 32GB 3200MHz
- **Storage:** 1TB M.2 NVMe SSD
- **GPU:** NVIDIA GTX 1070 Ti
- **Purpose:** General purpose computing and development

### Raspberry Pi 4B

- **Processor:** ARM Cortex-A72 (4 cores)
- **RAM:** 4GB/8GB
- **Storage:** MicroSD Card
- **Purpose:** Lightweight services and edge computing
